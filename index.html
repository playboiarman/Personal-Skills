How I Started Coding

I have always been fascinated by technology and computer science. My journey began when I started working on small coding projects with my brother. One of my first projects was creating a fake restaurant order system in Python using strings and integers. It was a simple but exciting step into the world of coding.

Over time, I explored multiple programming languages, including HTML, JavaScript, Java, Python, and C++. Each language has given me unique insights and skills, and I enjoy applying them to build solutions and explore creativity in software development.
