Skill Details

Explore the programming languages I've learned and how they help me solve real-world problems.

HTML

HTML (HyperText Markup Language) is the foundation of all web pages. I learned HTML to create websites effectively, ensuring clean and readable code.

  • Techniques: Semantic tags, forms, and tables.
  • Benefits: Ensures clean website structure and accessibility.
  • Projects: Basic personal websites and landing pages.

JavaScript

JavaScript gave me the power to make web pages interactive and dynamic. From DOM manipulation to API integration, it’s one of my favorite languages.

  • Techniques: DOM manipulation, functions, and event handling.
  • Benefits: Adds interactivity to websites and improves user experience with a nice UI.
  • Projects: Interactive forms, dynamic menus, and simple games.

Python

Python is where my coding journey started. Its simplicity and versatility allowed me to focus on logic rather than syntax.

  • Techniques: Loops, conditionals, and basic data structures.
  • Benefits: Perfect for problem-solving and automation.
  • Projects: Restaurant ordering systems, calculators, and data scripts.

Java

I learned Java to understand Object-Oriented Programming (OOP). It taught me how to create clean and reusable code for larger projects.

  • Techniques: Classes, objects, and inheritance.
  • Benefits: Great for building scalable and secure applications.
  • Projects: Basic games, console apps, and small tools.

C++

C++ helped me understand low-level programming concepts and improved my performance optimization skills.

  • Techniques: Pointers, memory management, and OOP principles.
  • Benefits: High-performance programs and efficiency.
  • Projects: Basic algorithms, console games, and data structures.
